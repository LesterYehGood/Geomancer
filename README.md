### 鄉民風水師
[Google Play 下載](https://play.google.com/store/apps/details?id=tacoball.com.geomancer)

這是一個利用資料視覺化的方式設計的人文地理工具，目前可查詢：
* 全台灣凶宅地點
* 台北市勞工局違反勞基法事業地點
